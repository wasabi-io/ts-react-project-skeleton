#### Getting Started

* Before Start Project

```ssh
$ git clone ....`
$ cd ts-react-project-skeleton
$ npm install
```

* run start (Development Mode) 
```ssh
$ npm start
```

* run test (Test Mode) 

```ssh
$ npm test
```

*  build project (Production Mode) 
```ssh
$ npm start
```

* run coverage

```ssh
$ npm run coverage
```

* generate api docs

```ssh
$ npm run docs
```

* build library as javascript (common-js)

```ssh
$ npm build
```
